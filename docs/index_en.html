🎬 iAnchor

Turn a topic + data into a vertical video with digital human, subtitles, and animated cards — in 3 minutes. Zero editing skills.

🍎 macOS M1-M4 🐍 Python 3.13 🤖 DeepSeek LLM 🗣️ Edge TTS 👄 MuseTalk Lip Sync 🧑 AI Avatar Gen
⭐ Star on GitHub

✨ Why iAnchor?

✍️

AI Scripting

Topic + data in, fluent narration out. Cloud API or local Ollama, your choice.

🗣️

Natural Voice

Microsoft Edge TTS with 16 voices. Smooth, natural pronunciation.

👄

Lip Sync

MuseTalk audio-driven. Mouth movements precisely match speech.

📊

Dynamic Cards

Data, charts, and text auto-switch with narration. 4 animation engines to choose from.

📝

Sentence Subtitles

Gold text on rounded pill background. Split by punctuation. One per sentence.

🧑

AI Anchor Avatar

No photo needed — generate an HD anchor portrait from your script. Local Z-Image-Turbo or cloud API, manual upload also supported.

⚡

Fully Automatic

Script → TTS → Lip Sync → Cards → Subtitles → Final Video. One click, full pipeline.
