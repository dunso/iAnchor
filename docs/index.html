🎬 iAnchor

输入主题 + 素材，3 分钟后拿到一条带数字人、字幕、数据卡片的竖屏口播视频。全程 AI 驱动，零剪辑经验。

🍎 macOS M1-M4 🐍 Python 3.13 🤖 DeepSeek LLM 🗣️ Edge TTS 👄 MuseTalk 唇形 🧑 AI 形象生成
⭐ GitHub 仓库

✨ 为什么选 iAnchor？

✍️

AI 写稿

给个主题 + 数据，LLM 自动扩写成 3 分钟口播稿。云端 API / 本地 Ollama 随心切换。

🗣️

真人级语音

微软 Edge TTS 男声新闻腔，发音自然流畅。16 种音色可选，还能试听。

👄

唇形同步

MuseTalk 音频驱动，嘴型和声音精准对齐，数字人开口说话。

📊

数据卡片

价格、涨跌幅、成交量随口播自动切换。4 种动画引擎可选：PPT 卡片 / Manim / SD 文生图 / Remotion。

📝

逐句字幕

金黄字体 + 圆角背景 + 按标点拆分。一句一条，自动同步音频。

🧑

AI 主播形象

不用找照片，根据文案一键生成高清主播形象。本地 Z-Image-Turbo / 云端 API 双引擎，也支持手动上传。

⚡

全自动

LLM 写稿 → TTS 语音 → 唇形驱动 → 动画卡片 → 字幕合成 → 最终视频。一条命令，全链路自动。

🚀 5 步上手

1

克隆安装

git clone → bash install.sh

2

配置 Key

编辑 config.yaml 填入 DeepSeek API Key

3

选形象

上传照片，或 AI 一键生成主播形象

4

启动

bash start_ui.sh

5

生成

填主题+数据 → 一键出片

$ bash install.sh # 一键安装
$ bash start_ui.sh # 启动 WebUI
$ open http://127.0.0.1:7860
